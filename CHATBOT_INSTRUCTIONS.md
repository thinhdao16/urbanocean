# SCATERS UK ROADSHOW CHATBOT - USER GUIDE

## Running the Chatbots

**Original Version (Workshop):**
```bash
python chatbot_original.py
```
Simple keyword-matching chatbot for basic demonstration.

**Enhanced Version (Recommended):**
```bash
python chatbot_enhanced.py
```
Full-featured chatbot with comprehensive roadshow information.

## Commands & Prompts

**Quick Commands:**
- `cities` - View all roadshow cities and dates
- `london`, `birmingham`, `sheffield` - City-specific details
- `raptor` - The Raptor skateboard specifications
- `evening` - VIP evening events information
- `register` - Registration instructions
- `promotions` - Exclusive gear and rewards
- `help` - Show all commands
- `quit` - Exit chatbot

**Natural Language Questions:**
Ask naturally: "When is the London event?", "What skateboard are you launching?", "How do I register?", "What can I win?"

## Knowledge Base

**Roadshow Tour:** 3 cities (London Apr 4, Birmingham Apr 11, Sheffield Apr 18) with locations, activities, and promotions. Top 50 early registrants receive exclusive collaboration gear.

**The Raptor Skateboard:** 7-ply Canadian maple with carbon fiber, teal wave graphics, long-distance carving trucks, "No Beach? No Bother." grip tape. Launch: Spring 2026.

**VIP Evening Events:** Meal with free drinks, Skateboard England talk, meet team member. Entry via email campaign or website draw. Main entrant 18+ or with parent/guardian.

**Registration:** Free online registration. Under 18 requires parental consent. All participants receive Sunscreen Wax.

## Limitations

**Original:** Basic keyword matching, limited responses, educational demonstration only.

**Enhanced:** Text-based (no voice), pre-programmed knowledge (not AI-learning), cannot process registrations directly, no real-time updates. Requires Python 3.6+.

## Technical Requirements
Python 3.6+, no external dependencies, works on Windows/Mac/Linux, terminal access required.
